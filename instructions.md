# How to configure this repo

1. Use project [Link to project]() to generate certificates hash table
2. Make a pull request to this project using function [update_table()]()
3. Test the API

## Configuring GH pages

1. Go to the main page of this project on GitHub, you need to have admin access
2. Make sure the repo is public
3. Go to the "Settings" tab
4. Scroll down to the "GitHub Pages" section
5. Set "Source" to "master branch"
6. Wait a few minutes ans check if the pages below are ready

Go to: 
* https://cvbioinfoufmg.github.io/certificates/
* https://cvbioinfoufmg.github.io/certificates/2020/
